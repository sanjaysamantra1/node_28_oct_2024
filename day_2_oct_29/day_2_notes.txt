How to Run Javascript code
==========================
1. Javascript on Browser (FrontEnd)
  a. create a javascript file and add some code
  b. create an HTML file and include that javascript 
     <script src='abc.js' />
  c. run the HTML, along with HTML , javascript code gets executed
     (vscode extensions - liveserver , open in browser(alt+b))
   

2. Javascript on NodeJS
   a. create a javascript file and add some code
   b. make sure nodejs is installed
   c. run the javascript using nodejs
      node demo_1.js
